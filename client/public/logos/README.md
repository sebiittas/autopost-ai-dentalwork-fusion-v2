# Carpeta de Logos DentalWork

Coloca aquí los archivos de logo en formato PNG, SVG o WebP.

## Logos actuales esperados:
- logo-simbolo-dorado.png   → Símbolo dorado (marca de agua)
- logo-horizontal-blanco.png → Logo completo blanco (sobre fondos oscuros)
- logo-horizontal-negro.png  → Logo completo negro (sobre fondos claros)
- logo-isotipo.png           → Solo el ícono sin texto

## Cómo agregar un logo nuevo:
1. Pon el archivo en esta carpeta
2. Abre PromptStudio.jsx
3. Agrega un objeto al array LOGOS_CONFIG:
   { id: 'mi-logo', label: 'Mi Logo', file: '/logos/mi-logo.png', usage: 'Descripción de uso', position: 'bottom-right' }
